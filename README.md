Setup Instructions-To run this project, first clone the repository using "git clone https://github.com/prachithakur23567/e-commerceStore.git"  . Then install all dependencies using npm install. After that, start the project using npm start.
Assumptions made- First assumption is that the filter and the menu button (on the left side of the navbar) is used to toggle the filter section. Second Assumption Product Searching is based on Product Category and Product Name. Third Assumption is that filtering of brands is based on the filtering of category of products.
Architectural decisions: Have Made 3 Hooks useEffects, useFetch and userProduct to increase code reusability, I have used deboucing in the searching to get efficient search result and avoid unnecessary api calls. I have used useMemo to avoid unnecessary filtering.
